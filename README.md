# Course DAT251

This web application is created for teaching course DAT251. 

# Demo 

https://dat251.netlify.app/

# Get started using `npm`
1. Clone the repository locally.
2. Do `npm install`
3. Run `npm run test`


# Get started using `yarn`
1. Clone the repository locally.
2. Do `yarn`
3. Run `yarn test`
